# Personal-Finance-Analytic
Personal Finance Monitoring Dashboard is an interactive Power BI solution that tracks income, expenses, savings, and investment patterns to help individuals make smarter financial decisions. It provides actionable insights into spending behavior, savings growth, and financial health through dynamic visualizations and KPI tracking.
# Project Overview
The Personal Finance Monitoring Dashboard is a Power BI project designed to provide a complete view of personal financial performance. The dashboard analyzes income, expenses, savings, investments, spending categories, and financial trends to help users effectively manage their finances and achieve savings goals.
Through interactive visualizations and KPI monitoring, users can identify spending patterns, track savings progress, evaluate income sources, and make informed financial decisions.
# Business Objective
-To monitor and analyze personal financial data, identify spending habits, improve savings strategies, and support better financial planning through data-driven insights.
# Key Performance Indicators (KPIs)
- Total Income: 2M
- Total Savings: 326K
- Total Expenses: 1M
- Savings Rate: 21.59%
- Expense Ratio: 78.41%
- Analysis Period: 2018–2021
## Dashboard Insights
# Income Analysis
-Salary is the primary source of income.
-Additional income sources contribute a smaller share of total earnings.
# Expense Analysis
- House Rent is the highest expense category.
- Groceries, EMIs, and Healthcare account for a significant portion of monthly spending.
- Leisure and Shopping represent discretionary expenses.
# Savings Analysis
- Mutual Funds contribute the highest share of savings.
- Fixed Deposits and Emergency Funds support long-term financial stability.
- Liquid Cash provides short-term financial flexibility.
#Trend Analysis
- Income, expenses, and savings increased steadily from 2018 to 2020.
- Financial activity declined in 2021 compared to previous years.
- Savings growth reflects improved financial planning and investment habits.
